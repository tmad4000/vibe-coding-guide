[[How to make a site like jacobcole.net](http://jacobcole.net) or montykosma.com](http://montykosma.com) in 5 min. This process is tremendously life-organizing/inspiring and builds you a minimum viable web-presence.

[1. Copythis google doc](https://docs.google.com/document/d/1aZ2R1efxVcYUhvH-t2wXSj7YBjZzUJeGgBODOFWNh14/edit)by going to File > Make a Copy. Edit it/customize for yourself, including title. Don’t delete headings for things you intend to do in future, e.g. music.

2. File >  Publish to the Web > Start Publishing. Copy the URL of the new published doc (Document Link)

[[3. Get a free domain (extension .tk is free) for your homepage by going to](http://my.dot.tk/cgi-bin/login01.taloha)dot.tk](http://my.dot.tk/cgi-bin/login01.taloha) > sign in with Google > Domain Panel > Add Domain. Enter yourhomepage.tk. On next page paste in the URL of the doc. Register for 12 months. You need to renew every year (free) 1 month or so in advance

[[4. A](http://my.dot.tk/cgi-bin/login01.taloha)dd homepage to the list minimalisthomepages.com](http://minimalisthomepages.com)

[Protip #2: make a workflowy](https://workflowy.com/)and use it to start thoughtstreaming/taking notes/logging favorite quotes.
